# HR Analytics Dashboard - Tableau

## Project Overview
This project is an interactive HR Analytics Dashboard created using Tableau.

The dashboard analyzes:
- Employee attrition
- Salary distribution
- Age demographics
- Job satisfaction
- Department-wise attrition

## Tools Used
- Tableau Public
- Excel/CSV Dataset

## Dashboard Features
- KPI Cards
- Bar Charts
- Pie Charts
- Histogram
- Heatmap
- Interactive Filters

## Dataset
IBM HR Analytics Employee Attrition Dataset
